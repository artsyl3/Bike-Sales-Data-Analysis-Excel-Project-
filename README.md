📌 Project Overview

This project analyzes a Bike Sales dataset from Europe using Microsoft Excel.
The goal of the project is to explore customer demographics and purchasing behavior to understand what factors influence bike purchases.

The analysis includes data cleaning, feature creation, pivot tables, and building an interactive Excel dashboard.


📊 Dataset Information

The dataset contains information about customers and whether they purchased a bike.

Key Columns
Column	Description
ID	Unique customer identifier
Marital Status	Married or Single
Gender	Customer gender
Income	Customer yearly income
Children	Number of children
Education	Education level
Occupation	Job type
Home Owner	Whether the customer owns a house
Cars	Number of cars owned
Commute Distance	Distance to work
Region	Customer region
Age	Customer age
Purchased Bike	Whether the customer purchased a bike

🧹 Data Cleaning (Excel)

Several data cleaning steps were performed to prepare the dataset for analysis:

1. Removed Duplicate Records

Duplicate rows were removed to ensure each customer is counted once.

2. Standardized Values

Short values were replaced with clearer labels:

Original	Updated
M	Married
S	Single
3. Created Age Groups

A new column Age Bracket was created to categorize customers:

Adolescent

Middle Age

Old

This helps analyze purchasing patterns across age groups.

📊 Data Analysis

Using Pivot Tables, several insights were explored.

1. Income vs Bike Purchase

Analyzed how income affects the likelihood of buying a bike.

Finding:
Customers with higher average income purchased more bikes.

2. Commute Distance vs Bike Purchase

Explored whether commuting distance affects bike purchases.

Finding:
Customers with shorter commute distances tend to buy more bikes.

3. Age Group vs Bike Purchase

Analyzed purchasing patterns across age groups.

Finding:
Middle-aged customers purchase the most bikes.
